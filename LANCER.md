# 🔮 Oracle Tarot Professionnel

## Installation
```bash
pip install streamlit
```

## Lancement
```bash
cd tarot_oracle
streamlit run interface/app_final.py
```

## Contenu de la banque
- 22 Arcanes Majeurs × 5 secteurs × 10 variantes = 1100 blocs
- 14 Coupes × 5 secteurs × 5 variantes = 350 blocs
- 14 Bâtons × 5 secteurs × 5 variantes = 350 blocs
- + Épées et Pentacles en cours d'expansion

## Utilisation
1. Renseigne le prénom du client et sa question
2. Choisis le secteur (amour / travail / argent / famille / spirituel)
3. Sélectionne le type de tirage (1, 3, 5 ou 10 cartes)
4. Coche les cartes tirées (et si renversées)
5. Clique "GÉNÉRER"
6. Copie le discours et envoie au client
